# Phase 2: Data Preprocessing

## Overview
In this phase, the cleaned dataset was prepared for model training.

## Steps Performed
- Loaded cleaned dataset
- Checked for missing values and duplicates
- Encoded categorical variable (congestion level)
- Split dataset into features and target
- Applied feature scaling:
  - MinMaxScaler
  - StandardScaler
- Performed train-test split (80:20)

## Outcome
- Clean and normalized dataset ready for model training
