# Phase 1: Data Collection and EDA

## Overview
In this phase, a synthetic dataset for WiFi congestion was generated using NumPy.

## Dataset Features
- Number of devices
- Latency
- Packet loss
- Signal strength
- Bandwidth
- Router load
- And other network parameters

## Steps Performed
- Data generation using NumPy
- Created dataset using Pandas DataFrame
- Checked shape, info, and statistical summary
- Checked missing values
- Removed outliers using IQR method

## Visualization
- Histogram plots
- Scatter plots
- Correlation matrix
- Distribution of congestion levels

## Output
- wifi_raw_data.csv (raw dataset)
- cleaned_wifi_data.csv (after cleaning)
