# Phase 4: Model Evaluation

## Overview
In this phase, the performance of the trained model was evaluated.

## Evaluation Metrics Used
- Accuracy
- Confusion Matrix
- Precision
- Recall
- F1-score

## Results
- The model achieved good accuracy
- All classes (Low, Medium, High) were predicted
- Some misclassification occurred in Medium class

## Interpretation
- The model performs well overall
- Slight improvement can be done for Medium class prediction

## Conclusion
The model is reliable and suitable for predicting WiFi congestion levels.
